# SPFx Hide SharePoint Bar

This project is a SharePoint Framework (SPFx) Application Customizer that removes the default SharePoint top bar to create a cleaner and more focused user interface.

## Overview

This solution was built to improve the visual experience of SharePoint pages by removing the default SharePoint header. It allows organizations to create more customized, application-like interfaces without the distraction of standard navigation elements.

## Key Features

- Removes the default SharePoint top navigation bar  
- Creates a cleaner and more focused page layout  
- Enhances the ability to build custom UI experiences  
- Lightweight and easy to deploy  

## Tech Stack

- SharePoint Framework (SPFx)  
- TypeScript  

## Getting Started

npm install  
gulp serve  

## Build and Package

gulp bundle --ship  
gulp package-solution --ship  

## Deployment

1. Upload the .sppkg file to the SharePoint App Catalog  
2. Deploy the solution  
3. Add the extension to your site
